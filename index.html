Masaki

Created : Monday Feburary 5th 2018
Developer : Revolx#0001
Version : 1.1.0 (Discontinued)
Coded in JavaScript using Discord.js

Masaki is a Discord Bot which was created for a challenge and has slowly developed to greater lengths since the first server joined. Originally coded on Discord.js Stable but has now transitioned to Discord.js Master.

  • Music
